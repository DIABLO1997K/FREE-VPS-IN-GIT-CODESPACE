## ⚒️ SETUP
First Fork this repo make it private or public its your choice (private recommended).
Then go to your main git page.  
**Step 1:** Click on Green🟢 button named "CODE" Then Click on "CODESPACE".  
**Step 2:** After Codespace is ready just open terminal and paste the following commands.  
```
curl -O https://raw.githubusercontent.com/Safir-Akhtar/FREE-VPS-IN-GIT-CODESPACE/refs/heads/main/install.sh
chmod +x install.sh
./install.sh
```
**Step 3:** Follow Terminal. 
Now you are good to go🎉🎉
